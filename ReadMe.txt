What is a ZPM file?:
A ZPM file, known as Zip Prime MegaPlasma file, Is a Customly Formatted WinRar Achive file that has same functions as the '.rar' one, but is protected with a password for safety reasons, and has a '.ZPM' extension in it. It is used to archive the files to back it up into a single file, so that the backing up process is within 1 file.

How do you use a ZPM File?:
1. Define it to the WinRar software
2. Enter in the Password
